# Nó Wizebot

<img className="block dark:hidden" src="/images/wizebot-logo-light.png" alt="Logo Wizebot" />
<img className="hidden dark:block" src="/images/wizebot-logo-dark.png" alt="Logo Wizebot" />

O nó Wizebot permite integrar a API WhatsApp da Wizebot com seus fluxos de trabalho no n8n, possibilitando enviar mensagens, gerenciar assinantes, trabalhar com etiquetas e muito mais.

## Pré-requisitos

- Uma conta Wizebot com acesso à API WhatsApp
- Um token de API válido da Wizebot

## Instalação

Você pode instalar este nó comunitário usando o npm:

```bash
npm install n8n-nodes-wizebot
```

Após a instalação, reinicie seu servidor n8n para que o nó seja reconhecido.

## Configuração

Para usar o nó Wizebot, você precisa configurar as credenciais da API:

1. No n8n, vá para **Configurações** > **Credenciais** > **Criar Nova**
2. Selecione **API Wizebot**
3. Insira seu token de API da Wizebot
4. Salve as credenciais

<img src="/images/wizebot-credentials.png" alt="Configuração de Credenciais" />

## Operações Disponíveis

O nó Wizebot oferece as seguintes operações:

### Enviar Mensagem

Envie uma mensagem de texto para qualquer número de celular via WhatsApp.

<img src="/images/wizebot-enviar-mensagem.png" alt="Enviar Mensagem" />

**Parâmetros:**

| Parâmetro | Descrição |
|-----------|-----------|
| ID do Número de Telefone | ID do número de telefone da conta do WhatsApp |
| Mensagem | Conteúdo da mensagem de texto |
| Número de Telefone | Número do destinatário (com código do país) |

**Exemplo de Resposta:**

```json
{
  "status": "1",
  "message": "Message sent successfully."
}
```

## Exemplos de Uso

### Enviar mensagem de boas-vindas para novos contatos

<img src="/images/wizebot-exemplo-boas-vindas.png" alt="Exemplo de Boas-vindas" />

Este workflow envia uma mensagem de boas-vindas para novos contatos adicionados ao seu WhatsApp:

1. Use um trigger de "Webhook" para receber notificações de novos contatos
2. Use o nó Wizebot com a operação "Enviar Mensagem" para enviar uma mensagem de boas-vindas
3. Opcionalmente, adicione o contato a uma etiqueta "Novo Cliente" usando a operação "Atribuir Etiquetas"

### Automação de atendimento ao cliente

<img src="/images/wizebot-exemplo-atendimento.png" alt="Exemplo de Atendimento" />

Este workflow automatiza o atendimento ao cliente:

1. Use um trigger de "Webhook" para receber mensagens de clientes
2. Use um nó "Switch" para analisar o conteúdo da mensagem
3. Para perguntas comuns, use o nó Wizebot com a operação "Enviar Mensagem" para responder automaticamente
4. Para consultas complexas, use a operação "Atribuir Chat à Equipe" para encaminhar para um atendente

## Solução de Problemas

### Mensagens não estão sendo enviadas

- Verifique se o token da API está correto e ativo
- Confirme se o número de telefone do destinatário está no formato correto (com código do país)
- Verifique se você não excedeu o limite de assinantes da sua conta

### Erros de autenticação

- Verifique se o token da API está correto nas credenciais
- Confirme se sua conta Wizebot está ativa e com pagamentos em dia

### O nó não aparece no n8n

- Certifique-se de que o pacote foi instalado corretamente
- Reinicie o servidor n8n após a instalação
- Verifique os logs do n8n para possíveis erros
