# Como Fazer Integração com a Kirvano

Saiba como utilizar nossas notificações para se comunicar com sistemas externos.

## O Que é Webhook?

Webhook é uma forma de recebimento de informações que são passadas quando um evento acontece. Na prática, é a forma de receber informações entre dois sistemas de uma forma passiva.

Além disso, o webhook é uma maneira prática para um app ou sistema fornecer outras aplicações com informações em tempo real.

No vídeo abaixo, você vai aprender a integrar a Kirvano com a Wizebot utilizando webhooks. Vamos mostrar passo a passo como configurar essa integração para enviar notificações e se comunicar de forma eficiente entre os sistemas.

- Passo a passo da configuração inicial
- Definição dos eventos que irão disparar notificações

### Envio de Notificações
