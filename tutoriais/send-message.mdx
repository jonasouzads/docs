## Tipos de Mensagem

a Wizebot suporta vários tipos de mensagem:

  <Card title="Texto" icon="message">
    Mensagens de texto simples ou formatadas
  </Card>
  <Card title="Mídia" icon="image">

  </Card>
  <Card title="Botões" icon="circle-dot">
    Mensagens interativas com botões
  </Card>
  <Card title="Listas" icon="list">

## Enviando Mensagem de Texto

```javascript
const response = await fetch('https://app.wizebot.com.br/api/v1/whatsapp/send', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/x-www-form-urlencoded',
    'Authorization': 'Bearer SEU_TOKEN'
  },
  body: new URLSearchParams({
    'phone_number': '5511999999999',
    'message': 'Olá! Esta é uma mensagem de teste.'
  })
});
```

## Enviando Mídia

Para enviar mídia, você precisa fornecer a URL do arquivo:

```javascript
const response = await fetch('https://app.wizebot.com.br/api/v1/whatsapp/send', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/x-www-form-urlencoded',
    'Authorization': 'Bearer SEU_TOKEN'
  },
  body: new URLSearchParams({
    'phone_number': '5511999999999',
    'message': 'Confira nossa promoção!',
    'media_url': 'https://exemplo.com/imagem.jpg'
  })
});
```
