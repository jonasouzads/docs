# Atualizar Assinante

Esta API permite atualizar as informações de um assinante existente na sua conta do WhatsApp Business.

## Endpoint da API

<ParamField method="POST" type="string">
  Atualizar as informações de um assinante existente na sua conta do WhatsApp
</ParamField>

## Parâmetros

<ParamField query="apiToken" type="string" required>
  Sua chave de API
</ParamField>

<ParamField query="phone_number_id" type="string" required>
  Número de telefone da conta do WhatsApp ID
</ParamField>

<ParamField query="phone_number" type="string" required>
  Número de telefone do assinante
</ParamField>

<ParamField query="first_name" type="string">
  Primeiro nome do seu assinante
</ParamField>

<ParamField query="last_name" type="string">
  Sobrenome do seu assinante
</ParamField>

<ParamField query="gender" type="string">
  Gênero do seu assinante
</ParamField>

<ParamField query="label_ids" type="string">
  Identificar IDs separados por vírgula, como (1,4,5)
</ParamField>

## Exemplo de Solicitação

<CodeGroup>
```bash POST
curl -X POST \ 
'https://app.wizebot.com.br/api/v1/whatsapp/subscriber/update' \
-d 'apiToken=API-KEY' \
-d 'phone_number_id=PHONE-NUMBER-ID' \
-d 'phone_number=PHONE-NUMBER' \
-d 'first_name=FIRST-NAME' \
-d 'last_name=LAST-NAME' \
-d 'gender=GENDER' \
-d 'label_ids=LABEL-IDS'
```
</CodeGroup>

## Exemplo de Resposta

<CodeGroup>
```json Sucesso
{
  "status": "1",
  "message": "Subscriber Updated Successfully."
}
```

```json Erro
{
  "status": "0",
  "message": "Subscriber not found."
}
```
</CodeGroup>
