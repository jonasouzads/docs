# Listar Etiquetas

Esta API permite obter uma lista de todas as etiquetas disponíveis na sua conta do WhatsApp Business.

## Endpoint da API

<ParamField method="POST" type="string">
  Obter lista de etiquetas da sua conta do WhatsApp
</ParamField>

## Parâmetros

<ParamField query="apiToken" type="string" required>
  Sua chave de API
</ParamField>

<ParamField query="phone_number_id" type="string" required>
  Seu número de telefone do WhatsApp
</ParamField>

## Exemplo de Solicitação

<CodeGroup>
```bash POST
curl -X POST \ 
'https://app.wizebot.com.br/api/v1/whatsapp/label/list' \
-d 'apiToken=API-KEY' \
-d 'phone_number_id=PHONE-NUMBER-ID'
```
</CodeGroup>

## Exemplo de Resposta

<CodeGroup>
```json Sucesso
{
  "status": "1",
  "message": [
    {
      "id": 1,
      "label_name": "Your label name",
      "status": "1"
    }
  ]
}
```

```json Erro
{
  "status": "0",
  "message": "Invalid API Key."
}
```
</CodeGroup>
