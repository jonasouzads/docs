# Atribuir Campos Personalizados

Este endpoint permite adicionar ou atualizar campos personalizados para um inscrito específico.

## Endpoint

```bash
POST https://app.wizebot.com.br/api/v1/whatsapp/subscriber/chat/assign-custom-fields
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| phone_number_id | String | Sim | ID do número de telefone da sua conta WhatsApp |
| phone_number | String | Sim | Número do telefone do inscrito |
| custom_fields | JSON | Sim | Objeto JSON com nome e valor dos campos personalizados |

## Exemplo de Requisição

### POST
```bash
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/subscriber/chat/assign-custom-fields' \
  -d 'apiToken=API-KEY' \
  -d 'phone_number_id=PHONE-NUMBER-ID' \
  -d 'phone_number=PHONE-NUMBER' \
  -d 'custom_fields={"cidade": "São Paulo", "interesse": "Marketing"}'
```

## Formato do custom_fields

```json
{
  "custom_filed_name1": "custom_field_value1",
  "custom_filed_name2": "custom_field_value2"
}
```

## Resposta

```json
{
  "status": "1",
  "message": "Alterações do inscrito foram salvas com sucesso."
}
```

## Observações

<Info>
- Os campos personalizados existentes serão atualizados se já existirem
- Novos campos serão criados se não existirem
- Você pode enviar múltiplos campos em uma única requisição
</Info>

<Warning>
- Os nomes dos campos não devem conter caracteres especiais
- O valor máximo para cada campo é de 255 caracteres
</Warning>
