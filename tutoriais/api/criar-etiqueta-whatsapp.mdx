# Criar Etiqueta

Esta API permite criar uma nova etiqueta na sua conta do WhatsApp Business.

## Endpoint da API

<ParamField method="POST" type="string">
  Criar uma nova etiqueta na sua conta do WhatsApp
</ParamField>

## Parâmetros

<ParamField query="apiToken" type="string" required>
  Sua chave de API
</ParamField>

<ParamField query="phone_number_id" type="string" required>
  Seu número de telefone do WhatsApp
</ParamField>

<ParamField query="label_name" type="string" required>
  Nome da etiqueta
</ParamField>

## Exemplo de Solicitação

<CodeGroup>
```bash POST
curl -X POST \ 
'https://app.wizebot.com.br/api/v1/whatsapp/label/create' \
-d 'apiToken=API-KEY' \
-d 'phone_number_id=PHONE-NUMBER-ID' \
-d 'label_name=LABEL-NAME'
```
</CodeGroup>

## Exemplo de Resposta

<CodeGroup>
```json Sucesso
{
  "status": "1",
  "message": "Label has been created successfully."
}
```

```json Erro
{
  "status": "0",
  "message": "Invalid API Key."
}
```
</CodeGroup>
