# Listar Assinantes

Esta API permite obter uma lista de assinantes da sua conta do WhatsApp Business.

## Endpoint da API

<ParamField method="POST" type="string">
  Obter lista de assinantes da sua conta do WhatsApp
</ParamField>

## Parâmetros

<ParamField query="apiToken" type="string" required>
  Sua chave de API
</ParamField>

<ParamField query="phone_number_id" type="string" required>
  Seu número de telefone do WhatsApp
</ParamField>

<ParamField query="limit" type="number" required>
  Obter número de assinantes
</ParamField>

<ParamField query="offset" type="number">
  Deslocamento de paginação
</ParamField>

## Exemplo de Solicitação

<CodeGroup>
```bash POST
curl -X POST \ 
'https://app.wizebot.com.br/api/v1/whatsapp/subscriber/list' \
-d 'apiToken=API-KEY' \
-d 'phone_number_id=PHONE-NUMBER-ID' \
-d 'limit=10' \
-d 'offset=1'
```
</CodeGroup>
