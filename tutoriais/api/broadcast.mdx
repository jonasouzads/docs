# Enviar Broadcast com Template

Este endpoint permite enviar mensagens em massa usando templates pré-aprovados.

## Endpoint

```bash
POST https://app.wizebot.com.br/api/v1/whatsapp/broadcast/template/send
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| phone_number_id | String | Sim | ID do número de telefone da sua conta WhatsApp |
| fluxo_de_dados | JSON | Sim | Dados do fluxo no formato JSON |

## Exemplo de Requisição

### GET
```bash
https://app.wizebot.com.br/api/v1/whatsapp/broadcast/template/send?apiToken=API-KEY&phone_number_id=PHONE-NUMBER-ID
```

### POST
```bash
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/broadcast/template/send' \
  -d 'apiToken=API-KEY' \
  -d 'phone_number_id=PHONE-NUMBER-ID'
```

## Resposta

```json
{
  "status": "1",
  "visual_flow_campaign_id": 1727,
  "message": "Dados salvos com sucesso.",
  "redirect_on_success_url": ""
}
```

## Observações

<Warning>
Certifique-se de que:
- O template usado está aprovado pelo WhatsApp
- Os números de telefone estão no formato correto
- Você tem saldo suficiente para o broadcast
</Warning>
