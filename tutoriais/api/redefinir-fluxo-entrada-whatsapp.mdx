# Redefinir Fluxo de Entrada do Usuário

Esta API permite redefinir o fluxo de entrada de um usuário específico no WhatsApp.

## Endpoint da API

<ParamField method="POST" type="string">
  Redefinir o fluxo de entrada de um usuário específico no WhatsApp
</ParamField>

## Parâmetros

<ParamField query="apiToken" type="string" required>
  Sua chave de API
</ParamField>

<ParamField query="phone_number_id" type="string" required>
  Número de telefone da conta do WhatsApp ID
</ParamField>

<ParamField query="phone_number" type="string" required>
  Número de telefone do assinante
</ParamField>

## Exemplo de Solicitação

<CodeGroup>
```bash POST
curl -X POST \ 
'https://app.wizebot.com.br/api/v1/whatsapp/subscriber/reset/user-input-flow' \
-d 'apiToken=API-KEY' \
-d 'phone_number_id=PHONE-NUMBER-ID' \
-d 'phone_number=PHONE-NUMBER'
```
</CodeGroup>

## Exemplo de Resposta

<CodeGroup>
```json Sucesso
{
  "status": "1",
  "message": "Reset User Input Flow successfully."
}
```

```json Erro
{
  "status": "0",
  "message": "Subscriber not found."
}
```
</CodeGroup>
