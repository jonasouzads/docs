# Atribuir Chat à Equipe

Esta API permite atribuir o chat de um assinante a um membro específico da equipe.

## Endpoint da API

<ParamField method="POST" type="string">
  Atribuir o chat de um assinante a um membro específico da equipe
</ParamField>

## Parâmetros

<ParamField query="apiToken" type="string" required>
  Sua chave de API
</ParamField>

<ParamField query="phone_number_id" type="string" required>
  Seu número de telefone do WhatsApp
</ParamField>

<ParamField query="phone_number" type="string" required>
  Número de telefone do assinante
</ParamField>

<ParamField query="team_member_id" type="integer" required>
  ID do Membro da Equipe
</ParamField>

## Exemplo de Solicitação

<CodeGroup>
```bash POST
curl -X POST \ 
'https://app.wizebot.com.br/api/v1/whatsapp/subscriber/chat/assign-to-team-member' \
-d 'apiToken=API-KEY' \
-d 'phone_number_id=PHONE-NUMBER-ID' \
-d 'phone_number=PHONE-NUMBER' \
-d 'team_member_id=TEAM-MEMBER-ID'
```
</CodeGroup>

## Exemplo de Resposta

<CodeGroup>
```json Sucesso
{
  "status": "1",
  "message": "Successfully Assign Agent to Conversation."
}
```

```json Erro
{
  "status": "0",
  "message": "Subscriber not found."
}
```
</CodeGroup>
