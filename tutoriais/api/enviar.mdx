# Enviar Mensagem

Este endpoint permite enviar mensagens de texto para qualquer número do WhatsApp.

## Mensagens de Texto vs Modelos

<Info>
Existem duas formas de enviar mensagens:

1. **Mensagens de Texto (Sessão)**: Podem ser enviadas apenas dentro de uma janela de 24 horas após a última mensagem do cliente.
2. **Mensagens de Modelo**: Podem ser enviadas a qualquer momento, mas precisam seguir um modelo pré-aprovado.
</Info>

## Endpoint

```bash
POST https://app.wizebot.com.br/api/v1/whatsapp/send
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| phone_number_id | String | Sim | ID do número de telefone da sua conta WhatsApp |
| message | String | Sim | Conteúdo da mensagem (use URL encode para GET) |
| phone_number | String | Sim | Número do destinatário (com código do país, apenas números) |

## Exemplo de Requisição

### GET
```bash
https://app.wizebot.com.br/api/v1/whatsapp/send?apiToken=API-KEY&phone_number_id=PHONE-NUMBER-ID&message=TEXT-MESSAGE&phone_number=PHONE-NUMBER
```

### POST
```bash
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/send' \
  -d 'apiToken=API-KEY' \
  -d 'phone_number_id=PHONE-NUMBER-ID' \
  -d 'message=TEXT-MESSAGE' \
  -d 'phone_number=PHONE-NUMBER'
```

## Respostas Possíveis

### Sucesso
```json
{
  "status": "1",
  "message": "Mensagem enviada com sucesso."
}
```

### Erro - Limite de Assinantes
```json
{
  "status": "0",
  "message": "Limite de assinantes excedido. Você não pode ter mais assinantes."
}
```
