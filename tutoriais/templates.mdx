## O que são Templates?

Templates são modelos de mensagem pré-aprovados que podem ser usados para iniciar conversas com seus clientes.

## Criando um Template

1. Acesse o painel da Wizebot
2. Vá para "Templates > Novo Template"
3. Escolha o tipo de template:
   - Texto simples
   - Botões
   - Lista
   - Mídia

<Note>
  Templates precisam ser aprovados pelo WhatsApp antes de serem utilizados.
</Note>

## Exemplo de Template

```javascript
const response = await fetch('https://app.wizebot.com.br/api/v1/whatsapp/send', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/x-www-form-urlencoded',
    'Authorization': 'Bearer SEU_TOKEN'
  },
  body: new URLSearchParams({
    'phone_number': '5511999999999',
    'template_name': 'boas_vindas',
    'language': 'pt_BR',
    'components': JSON.stringify([
      {
        'type': 'body',
        'parameters': [
          {
            'type': 'text',
            'text': 'João'
          }
        ]
      }
    ])
  })
});
```

## Variáveis Disponíveis

| Variável | Descrição |
|----------|-----------|
| {nome} | Nome do contato |
| {empresa} | Nome da empresa |
| {data} | Data atual |
