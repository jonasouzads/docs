## O que é a WhatsApp Cloud API?

A WhatsApp Cloud API é uma interface baseada em nuvem que permite que empresas se comuniquem com seus clientes através do WhatsApp de forma programática.

## Tipos de Mensagem Suportados

| Tipo | Descrição | Suporte |
|------|-----------|---------|
| Texto | Mensagens de texto simples | ✅ |
| Imagem | JPG, PNG | ✅ |
| Documento | PDF, DOC | ✅ |
| Áudio | MP3, OGG | ✅ |
| Vídeo | MP4, 3GPP | ✅ |
| Botões | Até 3 botões | ✅ |
| Lista | Menu de opções | ✅ |

## Janela de 24 Horas

<Note>
A WhatsApp Cloud API tem uma política de janela de 24 horas para respostas. Após esse período, apenas templates podem ser usados.
</Note>

## Limites e Restrições

- **Taxa de Envio**: Limite de mensagens por segundo
- **Tamanho de Mídia**: Máximo de 64MB
- **Templates**: Processo de aprovação necessário
- **Números**: Verificação de negócio requerida
