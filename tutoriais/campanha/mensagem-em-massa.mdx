# Como Fazer Disparo em Massa no WhatsApp

Nesse tutorial, você aprenderá como realizar disparos em massa no WhatsApp utilizando a API oficial da plataforma, garantindo segurança e evitando banimentos. O vídeo cobre todos os passos necessários para configurar e enviar uma campanha, desde a escolha da lista de contatos até o monitoramento dos resultados.

## Passo a Passo para Realizar Disparos em Massa

### 1. Importar Contatos e Criar Etiquetas

Antes de iniciar o disparo, certifique-se de que seus contatos estão importados na plataforma e devidamente etiquetados. Se você ainda não sabe como fazer isso, confira os tutoriais nos links abaixo:

- [Como importar contatos na Wizebot](/tutorials/contact-manager/import)
- [Como criar modelos de mensagem](/tutorials/message-templates/creation)

### 2. Acessar Campanhas

Na plataforma Wizebot, clique em "Campanhas" e selecione a opção de transmissão pelo WhatsApp.

### 3. Configurar a Campanha

1. Selecione a transmissão
2. Dê um nome à sua campanha
3. Defina o envio para "a qualquer momento" (caso seja para uma lista nova)
4. Escolha o modelo de mensagem já criado
5. Configure o botão de ação para direcionar seus leads para um fluxo de qualificação

### 4. Escolher Etiquetas e Excluir Contatos

1. Selecione a etiqueta da lista de contatos que receberá a mensagem
2. Use a opção "Excluir Etiqueta" para remover contatos específicos da campanha

### 5. Agendar o Envio

1. Defina o horário e o fuso horário da sua campanha
2. Crie uma etiqueta de teste com apenas o seu contato para verificar se tudo está funcionando

### 6. Monitorar o Envio

1. Acompanhe o status da campanha:
   - Total de mensagens enviadas
   - Mensagens entregues
   - Mensagens abertas
   - Contatos inacessíveis
2. Crie etiquetas automáticas para organizar contatos que não receberam a mensagem

## Conclusão

Agora que sua campanha está configurada, você pode realizar disparos em massa com segurança, usando a API oficial do WhatsApp e sem correr o risco de ser banido. Assista ao vídeo para ver o passo a passo completo:
