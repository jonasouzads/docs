## Visão Geral

Assinantes são todos os contatos que interagem com seu número do WhatsApp. a Wizebot oferece ferramentas poderosas para gerenciá-los.

## Adicionando Campos Personalizados

```javascript
const response = await fetch('https://app.wizebot.com.br/api/v1/subscribers/custom-fields', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/x-www-form-urlencoded',
    'Authorization': 'Bearer SEU_TOKEN'
  },
  body: new URLSearchParams({
    'phone_number': '5511999999999',
    'custom_fields': JSON.stringify({
      'ultima_compra': '2024-01-15',
      'valor_total': '1500.00',
      'categoria': 'VIP'
    })
  })
});
```

## Segmentação de Assinantes

Você pode segmentar seus assinantes por:

- Data de cadastro
- Última interação
- Tags atribuídas
- Campos personalizados
- Comportamento

## Exportação de Dados

Para exportar seus assinantes:

1. Acesse "Assinantes > Exportar"
2. Selecione os campos desejados
3. Escolha o formato (CSV ou Excel)
4. Clique em "Exportar"
