# Formulário de Captura - Automação de Leads

Neste tutorial, você aprenderá como criar um formulário de captura que envia automaticamente mensagens via WhatsApp para leads após o preenchimento do formulário. O passo a passo inclui a criação de um modelo de mensagem aprovado pelo WhatsApp, a configuração de webhook na plataforma Wizebot, e a integração do código em uma página usando Elementor ou outra plataforma. Você também verá como personalizar a mensagem com dados do lead e garantir que ela esteja em conformidade com as políticas do WhatsApp.

## Passo a Passo para Criar Formulário de Captura e Enviar Mensagens no WhatsApp

### 1. Acesso ao Gerenciador de Bots
- Acesse sua conta Wizebot
- Clique em "Gerenciador de Bots"
- Selecione o número de telefone desejado e clique em "Modelo de Mensagem"

### 2. Criação do Modelo de Mensagem
- Crie um modelo de mensagem preenchendo o texto
- Use variáveis para personalizar, como nome do lead
- Adicione botões de ação, como links para grupos ou fluxos de conversa
- Inclua a opção de "sair" no rodapé da mensagem para estar em conformidade com as políticas do WhatsApp
- Defina a categoria como "Utilidade" e salve

### 3. Aprovação e Sincronização
- Sincronize o modelo até que seja aprovado pelo WhatsApp
- Após a aprovação, use o modelo na automação

### 4. Configuração do Webhook
- Na Wizebot, crie um fluxo de trabalho e gere o link do webhook
- Adicione o link do webhook ao código do formulário no Elementor ou em outra plataforma

### 5. Integração do Formulário
- Cole o código do formulário e configure os campos, como nome, e-mail e WhatsApp
- O código trata as informações automaticamente para a API do WhatsApp

### 6. Teste e Finalização
- Teste o formulário preenchendo com seus dados e verifique se a mensagem é enviada
- A plataforma Wizebot captura as informações e permite mapeá-las para o fluxo

## Conclusão
