# Como Criar Fluxos de Conversa na Wizebot

Neste tutorial, você vai aprender passo a passo como criar fluxos de conversa automáticos na WizeBot. Descubra como configurar respostas automáticas para seus clientes com mensagens pré-configuradas, utilizando blocos interativos, mensagens condicionais e muito mais. Veja como integrar mídia, botões e até inteligência artificial para melhorar o atendimento ao cliente via WhatsApp.

## Passo a Passo para Criar Fluxos de Conversa

### 1. Acesse o Gerenciador de Bots
1. No painel da WizeBot, clique em "Gerenciador de Bot"
2. Escolha a sua conta
3. Clique na opção "Fluxo de Conversa"

### 2. Criação do Fluxo
1. Clique em "Criar" para abrir o construtor de fluxos
2. Selecione o tipo de mensagem que você deseja configurar:
   - Texto
   - Imagem
   - Áudio
   - Vídeo
   - Localização
   - Blocos interativos

### 3. Configuração de Blocos Interativos
1. Configure o bloco inicial com palavra-chave
2. Defina as ações que devem ser executadas
3. Adicione:
   - Botões
   - Cabeçalho
   - Rodapé
4. Defina os comandos a serem disparados ao clicar nos botões

### 4. Automatização com Condições
- Use campos customizados
- Configure etiquetas
- Defina mensagens personalizadas
- Configure remarketing baseado nas ações do usuário

### 5. Conexão com Outras Ferramentas
- Integre com Google Sheets para registrar informações
- Use API HTTP para requisições externas
- Conecte com Webhook para sincronização automática

### 6. Testar e Salvar
1. Clique em "Salvar" para finalizar as configurações
2. Teste o fluxo enviando a palavra-chave no WhatsApp
3. Verifique se todas as interações estão funcionando corretamente
