</Note>

## Problemas de Autenticação

<AccordionGroup>
  <Accordion title="API Key Inválida">
    - Verifique se a API Key está correta
    - Certifique-se de que a API Key não expirou
    - Confirme se você tem permissões suficientes
  </Accordion>

  <Accordion title="Erro de Autorização">
    - Confirme se sua conta está ativa
    - Verifique se o phone_number_id está correto
    - Certifique-se de que sua assinatura está em dia
  </Accordion>
</AccordionGroup>

## Problemas de Envio de Mensagem

<AccordionGroup>
  <Accordion title="Mensagem Não Entregue">
    - Verifique se o número do destinatário está correto
    - Confirme se o número está ativo no WhatsApp
    - Verifique se o formato do número está correto (com código do país)
  </Accordion>

  <Accordion title="Template Rejeitado">
    - Revise as diretrizes do WhatsApp
    - Verifique se o conteúdo está apropriado
    - Certifique-se de que o template está aprovado
  </Accordion>

  <Accordion title="Limite de Tempo Excedido">
    - Verifique a janela de 24 horas para mensagens
    - Use templates para mensagens fora da janela
    - Aguarde o cliente iniciar uma nova conversa
  </Accordion>
</AccordionGroup>

## Problemas Técnicos

<AccordionGroup>
  <Accordion title="Erro de Conexão">
    - Verifique sua conexão com a internet
    - Confirme se a API está online
    - Tente novamente em alguns minutos
  </Accordion>

  <Accordion title="Resposta Inválida">
    - Verifique o formato dos parâmetros
    - Confirme se todos os campos obrigatórios foram enviados
    - Revise a documentação da API
  </Accordion>

  <Accordion title="Rate Limit">
    - Verifique seus limites de requisição
    - Implemente um sistema de retry
    - Otimize suas chamadas à API
  </Accordion>
</AccordionGroup>

## Dicas de Resolução

1. **Logs Detalhados**
   - Mantenha logs de todas as requisições
   - Registre as respostas de erro
   - Monitore os tempos de resposta

2. **Testes**
   - Use o ambiente de sandbox quando disponível
   - Teste com diferentes cenários
   - Valide os formatos de entrada

3. **Melhores Práticas**
   - Siga as diretrizes do WhatsApp
   - Mantenha sua API Key segura
   - Implemente tratamento de erros

<Note>
  Se você não conseguir resolver o problema, entre em contato com nosso suporte técnico.
</Note>
