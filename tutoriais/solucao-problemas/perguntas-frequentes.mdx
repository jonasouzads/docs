# Perguntas Frequentes

Esta página contém respostas para as perguntas mais frequentes sobre a Wizebot.

## Geral

### O que é a Wizebot?
A Wizebot é uma solução que integra o WhatsApp API oficial para aumentar vendas e automatizar atendimento.

### A Wizebot é segura?
Sim, a Wizebot utiliza a API oficial do WhatsApp, garantindo total segurança e conformidade com as políticas do WhatsApp.

## Funcionalidades

### Como funciona a automação com IA?
A Wizebot utiliza inteligência artificial avançada (ChatGPT) para automatizar respostas e criar experiências personalizadas de atendimento.

### É possível fazer disparo em massa?
Sim, a Wizebot permite envio de mensagens em massa de forma segura e dentro das políticas do WhatsApp.
