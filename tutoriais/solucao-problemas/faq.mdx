## Geral

<AccordionGroup>
  <Accordion title="O que é a API do WhatsApp?">
    Nossa API do WhatsApp permite que você:
    - Envie mensagens programaticamente
    - Gerencie templates de mensagem
    - Receba webhooks de eventos
    - Automatize comunicações
  </Accordion>

  <Accordion title="Quais são os requisitos para usar a API?">
    Para começar, você precisa:
    - Uma conta ativa no Wizebot
    - Uma chave de API válida
    - Um número de WhatsApp Business
    - Templates aprovados (para mensagens iniciadas pela empresa)
  </Accordion>

  <Accordion title="Quanto custa usar a API?">
    - Consulte nossos planos de preços atuais
    - Os preços variam conforme o volume de mensagens
    - Há limites diferentes para cada plano
    - Entre em contato para planos personalizados
  </Accordion>
</AccordionGroup>

## Templates

<AccordionGroup>
  <Accordion title="Como criar um template?">
    1. Acesse o painel do Wizebot
    2. Vá para a seção de templates
    3. Clique em "Novo Template"
    4. Preencha as informações necessárias
    5. Envie para aprovação
  </Accordion>

  <Accordion title="Por que meu template foi rejeitado?">
    Motivos comuns incluem:
    - Conteúdo promocional inadequado
    - Linguagem não permitida
    - Formato incorreto
    - Violação das diretrizes do WhatsApp
  </Accordion>

  <Accordion title="Quanto tempo leva para aprovar um template?">
    - Geralmente 24-48 horas
    - Pode variar conforme o volume
    - Templates simples são mais rápidos
    - Rejeições podem atrasar o processo
  </Accordion>
</AccordionGroup>

## Mensagens

<AccordionGroup>
  <Accordion title="Posso enviar mensagens em massa?">
    - Sim, mas com limitações
    - Use templates aprovados
    - Respeite as políticas do WhatsApp
    - Evite spam ou conteúdo promocional excessivo
  </Accordion>

  <Accordion title="Como funciona a janela de 24 horas?">
    - Inicia quando o cliente envia uma mensagem
    - Permite respostas com texto livre
    - Após 24h, use apenas templates
    - O cliente pode reiniciar a janela
  </Accordion>

  <Accordion title="Quais tipos de mídia são suportados?">
    - Texto
    - Imagens
    - Vídeos
    - Documentos
    - Áudio
    - Localização
  </Accordion>
</AccordionGroup>

## Técnico

<AccordionGroup>
  <Accordion title="Como lidar com erros da API?">
    - Implemente retry com backoff
    - Monitore as respostas
    - Mantenha logs detalhados
    - Use nossos códigos de erro
  </Accordion>

  <Accordion title="Como configurar webhooks?">
    1. Configure uma URL HTTPS
    2. Adicione a URL no painel
    3. Implemente os handlers
    4. Teste com nosso sandbox
  </Accordion>

  <Accordion title="Qual é o rate limit da API?">
    - Varia por tipo de plano
    - Monitore headers de resposta
    - Implemente throttling
    - Contate suporte para aumentar limites
  </Accordion>
</AccordionGroup>
