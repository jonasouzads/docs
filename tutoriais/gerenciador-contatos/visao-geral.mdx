# Como Utilizar o Gerenciador de Contatos

No vídeo abaixo, você vai aprender a utilizar o Gerenciador de Contatos da Wizebot. Vamos explorar todas as funcionalidades do CRM, incluindo como ativar remarketing, utilizar etiquetas, importar contatos e gerenciar suas listas de contatos de forma eficiente.

## O Que Você Vai Aprender

1. **Ativação de Remarketing**
   - Como ativar campanhas de remarketing para atingir seus clientes

2. **Utilização de Etiquetas**
   - Como usar etiquetas para organizar e segmentar seus contatos

3. **Importação de Contatos**
   - Como importar suas listas de contatos para o sistema da Wizebot

4. **Gerenciamento de Contatos**
   - Dicas para gerenciar e manter suas listas de contatos atualizadas
