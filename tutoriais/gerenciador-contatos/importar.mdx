## Introdução

Importe sua lista de contatos para o WhatsApp em poucos passos: crie uma etiqueta para identificar a lista, prepare sua planilha e faça o upload.

## Passo 1: Criar uma Etiqueta

Antes de importar, crie uma etiqueta para marcar essa lista de contatos. Assim, na hora de disparar uma campanha, você seleciona a etiqueta e envia só para essa base.

1. Acesse **Configurações** > **Etiquetas** > **Nova Etiqueta**
2. Dê um nome para a etiqueta e, se quiser, escolha uma cor
3. Clique em **Criar**

## Passo 2: Preparar a Planilha

1. Crie uma coluna **telefone** com o número completo, incluindo o DDI do país (ex: `55` para o Brasil) — este é o único campo obrigatório
2. Crie uma coluna **nome** (opcional)
3. Salve o arquivo no formato **CSV separado por vírgulas**

<Tip>
No Google Planilhas: **Arquivo** > **Baixar** > **Valores separados por vírgula (.csv)**
</Tip>

Você também pode clicar em **Baixar modelo** na tela de importação para ver um exemplo pronto.

## Passo 3: Importar os Contatos

1. Na plataforma, acesse **Contatos** > **Opções** > **Importar Contatos**
2. Selecione o arquivo CSV da sua lista
3. Confira o mapeamento das colunas — a Wizebot detecta automaticamente colunas como telefone, nome, e-mail e observações
4. Selecione a etiqueta criada no Passo 1
5. Selecione a conta de WhatsApp para a qual os contatos serão importados
6. Clique em **Iniciar Importação**

<Note>
Quer importar mais de uma etiqueta por contato? Adicione uma coluna extra na planilha com os nomes das etiquetas separados por vírgula (ex: `etiqueta 01, etiqueta 02`). A Wizebot cria automaticamente cada etiqueta que ainda não existir.
</Note>

Ao concluir, atualize a tela de Contatos para ver a lista importada. Clique em um contato para conferir suas informações e etiquetas, ou use a etiqueta na hora de montar uma campanha.
