# API do WhatsApp Wizebot

A API do WhatsApp do Wizebot permite que você integre facilmente as funcionalidades do WhatsApp Business em sua aplicação. Com nossa API, você pode enviar mensagens, gerenciar contatos, criar campanhas de broadcast e muito mais.

## Autenticação

Todas as requisições à API precisam ser autenticadas usando um `apiToken`. Este token deve ser incluído em todas as chamadas à API.

```bash
apiToken: Sua-Chave-API
```

## Base URL

Todas as requisições devem ser feitas para a seguinte URL base:

```bash
https://app.wizebot.com.br/api/v1
```

## Formatos de Resposta

Todas as respostas da API são retornadas no formato JSON. Um exemplo de resposta bem-sucedida:

```json
{
  "status": "1",
  "message": "Operação realizada com sucesso"
}
```

Em caso de erro:

```json
{
  "status": "0",
  "message": "Mensagem de erro específica"
}
```

## Limites de Taxa

Para garantir a estabilidade do serviço, implementamos limites de taxa em nossas APIs:

- Máximo de 100 requisições por minuto por API Key
- Máximo de 1000 mensagens por dia por número de WhatsApp
- Máximo de 10.000 contatos por conta
