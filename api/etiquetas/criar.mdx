authentication: 'API Key'
---

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| phone_number_id | String | Sim | Seu número de telefone do WhatsApp |
| label_name | String | Sim | Nome da etiqueta |

## Exemplo de Requisição

```bash
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/label/create' \
  -H 'Content-Type: application/json' \
  -d '{
    "apiToken": "API-KEY",
    "phone_number_id": "PHONE-NUMBER-ID",
    "label_name": "LABEL-NAME"
  }'
```

## Exemplo de Resposta

```json
{
  "status": "1",
  "message": "Label has been created successfully."
}
```
