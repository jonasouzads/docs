description: 'Alterar status do pedido do catálogo'
api: 'POST https://app.wizebot.com.br/api/v1/whatsapp/catalog/order/status-change'
contentType: 'application/json'
authentication: 'API Key'
---

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| order_unique_id | String | Sim | Seu ID de Pedido do Catálogo |
| cart_status | String | Sim | O valor do status deve ser Aprovado\|Concluído\|Enviado\|Entregue\|Reembolsado |

## Exemplo de Requisição

```bash
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/catalog/order/status-change' \
  -H 'Content-Type: application/json' \
  -d '{
    "apiToken": "API-KEY",
    "order_unique_id": "ORDER-UNIQUE-ID",
    "cart_status": "CART-STATUS"
  }'
```

## Exemplo de Resposta

```json
{
  "status": true,
  "message": "Cart status has been updated successfully"
}
```

## Status Disponíveis

| Status | Descrição |
|--------|-----------|
| Aprovado | Pedido aprovado |
| Concluído | Pedido concluído com sucesso |
| Enviado | Pedido enviado |
| Entregue | Pedido entregue |
| Reembolsado | Pedido reembolsado |
