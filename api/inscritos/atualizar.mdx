---
title: 'Atualizar Assinante'
description: 'Atualizar um assinante existente'
api: 'POST https://app.wizebot.com.br/api/v1/whatsapp/subscriber/update'
contentType: 'application/json'
authentication: 'API Key'
---

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| phone_number_id | String | Sim | Número de telefone da conta do WhatsApp ID |
| phone_number | String | Sim | Número de telefone do assinante |
| first_name | String | Não | Primeiro nome do seu assinante |
| last_name | String | Não | Sobrenome do seu assinante |
| gender | String | Não | Gênero do seu assinante |
| label_ids | String | Não | Identificar IDs separados por vírgula, como (1,4,5) |

## Exemplo de Requisição

```bash
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/subscriber/update' \
  -H 'Content-Type: application/json' \
  -d '{
    "apiToken": "API-KEY",
    "phone_number_id": "PHONE-NUMBER-ID",
    "phone_number": "PHONE-NUMBER",
    "first_name": "FIRST-NAME",
    "last_name": "LAST-NAME",
    "gender": "GENDER",
    "label_ids": "LABEL-IDS"
  }'
```

## Exemplo de Resposta

```json
{
  "status": "1",
  "message": "Subscriber Updated Successfully."
}
