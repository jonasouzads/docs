---
title: 'Listar Assinantes'
description: 'Este endpoint retorna a lista de todos os assinantes do seu número do WhatsApp Business.'
---


## Endpoint

<Icon icon="circle-right" color="#34D399" size={20} /> **GET**

```bash
https://app.wizebot.com.br/api/v1/whatsapp/subscribers
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| phone_number_id | String | Sim | ID do número de telefone da conta WhatsApp |
| page | Integer | Não | Número da página (padrão: 1) |
| limit | Integer | Não | Quantidade de registros por página (padrão: 20) |
| search | String | Não | Termo para busca por nome ou número |

## Exemplo de Requisição

### <Icon icon="circle-right" color="#34D399" size={20} /> Requisição GET

```bash
https://app.wizebot.com.br/api/v1/whatsapp/subscribers?apiToken=CHAVE-DE-API&phone_number_id=ID-DO-NÚMERO&page=1&limit=20&search=João
```

## Exemplo de Resposta

```json
{
  "data": [
    {
      "id": "123456789",
      "name": "João Silva",
      "phone_number": "5511999999999",
      "status": "active",
      "tags": ["cliente", "vip"],
      "last_interaction": "2024-01-15T14:30:00Z",
      "custom_fields": {
        "cidade": "São Paulo",
        "idade": "32"
      }
    }
  ],
  "pagination": {
    "total": 150,
    "per_page": 20,
    "current_page": 1,
    "last_page": 8
  }
}
```

## Observações

- Os assinantes são ordenados por data da última interação
- O status pode ser: active, inactive, blocked
- As tags são personalizáveis e ajudam na segmentação
- Os custom_fields são campos personalizados definidos por você
- Use a paginação para lidar com grandes volumes de dados
