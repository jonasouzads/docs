# Listar Inscritos

Este endpoint retorna a lista de inscritos da sua conta do WhatsApp com suporte a paginação.

## Endpoint

```bash
GET https://app.wizebot.com.br/api/v1/whatsapp/subscriber/list
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| phone_number_id | String | Sim | ID do número de telefone da sua conta WhatsApp |
| limit | Number | Sim | Número de inscritos por página |
| offset | Number | Não | Número da página para paginação |

## Exemplo de Requisição

### GET
```bash
https://app.wizebot.com.br/api/v1/whatsapp/subscriber/list?apiToken=API-KEY&phone_number_id=PHONE-NUMBER-ID&limit=10&offset=1
```

### POST
```bash
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/subscriber/list' \
  -d 'apiToken=API-KEY' \
  -d 'phone_number_id=PHONE-NUMBER-ID' \
  -d 'limit=10' \
  -d 'offset=1'
```

    },
    {
      "subscriber_id": 157,

## Observações

<Info>
- Use os parâmetros limit e offset para implementar paginação
- O campo label_names retorna as etiquetas do inscrito separadas por vírgula
- Alguns campos podem retornar null se não estiverem preenchidos
</Info>
