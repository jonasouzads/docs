contentType: 'application/json'
authentication: 'API Key'
---

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| phone_number_id | String | Sim | Seu número de telefone do WhatsApp |
| phone_number | String | Sim | Número de telefone do assinante |
| custom_fields | JSON | Sim | Nome e valor de campos personalizados no formato JSON |

## Exemplo de Requisição

```bash
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/subscriber/chat/assign-custom-fields' \
  -H 'Content-Type: application/json' \
  -d '{
    "apiToken": "API-KEY",
    "phone_number_id": "PHONE-NUMBER-ID",
    "phone_number": "PHONE-NUMBER",
    "custom_fields": {
      "custom_filed_name1": "custom_field_value1",
      "custom_filed_name2": "custom_field_value2"
    }
  }'
```

## Exemplo de Resposta

```json
{
  "status": "1",
  "message": "Subscriber changes have been saved successfully."
}
