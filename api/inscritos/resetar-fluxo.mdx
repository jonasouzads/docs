## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| phone_number_id | String | Sim | Número de telefone da conta do WhatsApp ID |
| phone_number | String | Sim | Número de telefone do assinante |

## Exemplo de Requisição

```bash
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/subscriber/reset/user-input-flow' \
  -H 'Content-Type: application/json' \
  -d '{
    "apiToken": "API-KEY",
    "phone_number_id": "PHONE-NUMBER-ID",
    "phone_number": "PHONE-NUMBER"
  }'
```

## Exemplo de Resposta

```json
{
  "status": "1",
  "message": "Reset User Input Flow successfully."
}
```
