## Tipos de Template

<Tabs>
  <Tab title="Componentes">
    - **Header**: Título, imagem ou vídeo
    - **Body**: Texto principal com variáveis
    - **Footer**: Texto adicional opcional
    - **Buttons**: Botões interativos
  </Tab>
  <Tab title="Status">
    - **Approved**: Template aprovado e pronto para uso
    - **Pending**: Em análise pelo WhatsApp
    - **Rejected**: Reprovado, necessita ajustes
  </Tab>
</Tabs>

## Ponto de Extremidade da API

<Icon icon="circle-right" color="#F87171" size={20} /> **POST**

```bash
https://app.wizebot.com.br/api/v1/whatsapp/template/list
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| phone_number_id | String | Sim | Número de telefone da conta do WhatsApp ID |

## Exemplos de Código

<RequestExample>

```bash cURL
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/template/list' \
  -d 'apiToken=API-KEY' \
  -d 'phone_number_id=PHONE-NUMBER-ID'
```

```python Python
import requests

url = "https://app.wizebot.com.br/api/v1/whatsapp/template/list"
payload = {
    "apiToken": "API-KEY",
    "phone_number_id": "PHONE-NUMBER-ID"
}

response = requests.post(url, data=payload)
print(response.json())
```

```javascript JavaScript
fetch('https://app.wizebot.com.br/api/v1/whatsapp/template/list', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/x-www-form-urlencoded',
  },
  body: new URLSearchParams({
    'apiToken': 'API-KEY',
    'phone_number_id': 'PHONE-NUMBER-ID'
  })
})
.then(response => response.json())
.then(data => console.log(data));
```

```php PHP
<?php
$url = "https://app.wizebot.com.br/api/v1/whatsapp/template/list";
$data = array(
    'apiToken' => 'API-KEY',
    'phone_number_id' => 'PHONE-NUMBER-ID'
);

$options = array(
    'http' => array(
        'header'  => "Content-type: application/x-www-form-urlencoded\r\n",
        'method'  => 'POST',
        'content' => http_build_query($data)
    )
);

$context = stream_context_create($options);
$result = file_get_contents($url, false, $context);
echo $result;
?>
```

## Dicas e Boas Práticas

<AccordionGroup>
  <Accordion title="Componentes do Template">
    - Combine diferentes tipos de mídia no header
    - Use variáveis no corpo da mensagem
    - Adicione botões interativos quando necessário
    - Mantenha o footer conciso
  </Accordion>
  
  <Accordion title="Processo de Aprovação">
    - Siga as diretrizes do WhatsApp
    - Evite conteúdo promocional excessivo
    - Use linguagem clara e profissional
    - Teste o template antes de submeter
  </Accordion>

  <Accordion title="Gerenciamento">
    - Monitore o status dos templates
    - Atualize templates rejeitados
    - Mantenha um catálogo organizado
    - Documente as variáveis usadas
  </Accordion>
</AccordionGroup>
