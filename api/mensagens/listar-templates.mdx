## Tipos de Template

<Tabs>
  <Tab title="Componentes">
    - **Header**: Título, imagem ou vídeo
    - **Body**: Texto principal com variáveis
    - **Footer**: Texto adicional opcional
    - **Buttons**: Botões interativos
  </Tab>
  <Tab title="Status">
    - **Approved**: Template aprovado e pronto para uso
    - **Pending**: Em análise pelo WhatsApp
    - **Rejected**: Reprovado, necessita ajustes
  </Tab>
</Tabs>

## Endpoint da API

```bash
POST https://app.wizebot.com.br/api/v1/whatsapp/template/list
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| phone_number_id | String | Sim | Número de telefone da conta do WhatsApp ID |

## Testar a API

<ParamField body="apiToken" type="string" required>
  Sua chave de API
</ParamField>

<ParamField body="phone_number_id" type="string" required>
  Número de telefone da conta do WhatsApp ID
</ParamField>

<Tip>
  Use uma chave de API válida para listar todos os seus templates disponíveis.
</Tip>

## Exemplos

<CodeGroup>
```bash cURL
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/template/list' \
  -d 'apiToken=API-KEY' \
  -d 'phone_number_id=PHONE-NUMBER-ID'
```

```python Python
import requests

url = "https://app.wizebot.com.br/api/v1/whatsapp/template/list"
payload = {
    "apiToken": "API-KEY",
    "phone_number_id": "PHONE-NUMBER-ID"
}

response = requests.post(url, data=payload)
print(response.json())
```

```javascript JavaScript
fetch('https://app.wizebot.com.br/api/v1/whatsapp/template/list', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/x-www-form-urlencoded',
  },
  body: new URLSearchParams({
    'apiToken': 'API-KEY',
    'phone_number_id': 'PHONE-NUMBER-ID'
  })
})
.then(response => response.json())
.then(data => console.log(data));
```
</CodeGroup>
