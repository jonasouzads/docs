# Listar Templates

Este endpoint retorna todos os templates de mensagem disponíveis para sua conta do WhatsApp.

## Endpoint

```bash
GET https://app.wizebot.com.br/api/v1/whatsapp/template/list
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| phone_number_id | String | Sim | ID do número de telefone da sua conta WhatsApp |

## Exemplo de Requisição

### GET
```bash
https://app.wizebot.com.br/api/v1/whatsapp/template/list?apiToken=API-KEY&phone_number_id=PHONE-NUMBER-ID
```

### POST
```bash
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/template/list' \
  -d 'apiToken=API-KEY' \
  -d 'phone_number_id=PHONE-NUMBER-ID'
```
