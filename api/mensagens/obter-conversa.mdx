## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| phone_number_id | String | Sim | Seu número de telefone do WhatsApp |
| phone_number | String | Sim | Número de telefone do assinante |
| limit | Number | Sim | Buscar número da mensagem da conversa |
| offset | Number | Não | Deslocamento de paginação |

## Exemplo de Requisição

```bash
curl -X GET \
  'https://app.wizebot.com.br/api/v1/whatsapp/get/conversation?apiToken=API-KEY&phone_number_id=PHONE-NUMBER-ID&phone_number=PHONE-NUMBER&limit=10&offset=1'
```
