---
title: 'Enviar Broadcast'
description: 'Enviar mensagem em massa usando templates'
api: 'POST https://app.wizebot.com.br/api/v1/whatsapp/broadcast/send'
contentType: 'application/json'
authentication: 'API Key'
---

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| phone_number_id | String | Sim | Número de telefone da conta do WhatsApp ID |
| template_name | String | Sim | Nome do modelo a ser enviado |
| language | String | Não | Código do idioma do modelo (padrão: pt_BR) |
| phone_numbers | Array | Sim | Array de números de telefone que devem começar com o código do país |
| components | Array | Não | Array de componentes do modelo com valores dinâmicos |

## Exemplo de Requisição

```bash
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/broadcast/send' \
  -H 'Content-Type: application/json' \
  -d '{
    "apiToken": "API-KEY",
    "phone_number_id": "PHONE-NUMBER-ID",
    "template_name": "hello_world",
    "language": "pt_BR",
    "phone_numbers": ["5511999999999", "5511888888888"],
    "components": [
      {
        "type": "body",
        "parameters": [
          {
            "type": "text",
            "text": "John"
          }
        ]
      }
    ]
  }'
```

## Exemplo de Resposta

```json
{
  "status": "1",
  "message": "Broadcast message sent successfully."
}
