---
title: 'Disparar Bot'
description: 'Disparar fluxo do bot para qualquer número de celular'
api: 'POST https://app.wizebot.com.br/api/v1/whatsapp/bot/trigger'
contentType: 'application/json'
authentication: 'API Key'
---

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| phone_number_id | String | Sim | Número de telefone da conta do WhatsApp ID |
| phone_number | String | Sim | Deve começar com o código do país e apenas caracteres numéricos são permitidos |
| bot_id | Integer | Sim | ID do bot que será disparado |

## Exemplo de Requisição

```bash
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/bot/trigger' \
  -H 'Content-Type: application/json' \
  -d '{
    "apiToken": "API-KEY",
    "phone_number_id": "PHONE-NUMBER-ID",
    "phone_number": "PHONE-NUMBER",
    "bot_id": 123
  }'
```

## Exemplo de Resposta

```json
{
  "status": "1",
  "message": "Bot flow triggered successfully."
}
