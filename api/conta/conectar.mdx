# Conectar Conta do WhatsApp

Este endpoint permite conectar uma conta do WhatsApp Business à sua conta Wizebot.

## Endpoint

```bash
POST https://app.wizebot.com.br/api/v1/whatsapp/account/connect
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| user_id | Integer | Sim | ID do usuário proprietário da conta |
| whatsapp_business_account_id | String | Sim | ID da Conta WhatsApp Business |
| access_token | String | Sim | Token de acesso do WhatsApp |

## Exemplo de Requisição

### GET
```bash
https://app.wizebot.com.br/api/v1/whatsapp/account/connect?apiToken=API-KEY&user_id=USER-ID&whatsapp_business_account_id=WHATSAPP-BUSINESS-ACCOUNT-ID&access_token=ACCESS-TOKEN
```

### POST
```bash
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/account/connect' \
  -d 'apiToken=API-KEY' \
  -d 'user_id=USER-ID' \
  -d 'whatsapp_business_account_id=WHATSAPP-BUSINESS-ACCOUNT-ID' \
  -d 'access_token=ACCESS-TOKEN'
```

## Resposta

```json
{
  "status": "1",
  "message": "Conta do WhatsApp conectada com sucesso."
}
```
