---
title: 'Conectar Conta'
description: 'Conectar conta do WhatsApp Business'
api: 'POST https://app.wizebot.com.br/api/v1/whatsapp/account/connect'
contentType: 'application/json'
authentication: 'API Key'
---

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| user_id | Integer | Sim | ID do usuário do proprietário da conta do WhatsApp |
| whatsapp_business_account_id | String | Sim | ID da Conta do WhatsApp Business |
| access_token | String | Sim | Token de Acesso |

## Exemplo de Requisição

```bash
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/account/connect' \
  -H 'Content-Type: application/json' \
  -d '{
    "apiToken": "API-KEY",
    "user_id": "USER-ID",
    "whatsapp_business_account_id": "WHATSAPP-BUSINESS-ACCOUNT-ID",
    "access_token": "ACCESS-TOKEN"
  }'
```

## Exemplo de Resposta

```json
{
  "status": "1",
  "message": "Whatsapp account has been connect successfully."
}
```
