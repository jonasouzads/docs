---
title: 'Listar Templates'
description: 'Este endpoint retorna a lista de todos os templates de mensagem disponíveis para sua conta.'
---


## Endpoint

<Icon icon="circle-right" color="#34D399" size={20} /> **GET** 
<Icon icon="circle-right" color="#F87171" size={20} /> **POST**

```bash
https://app.wizebot.com.br/api/v1/whatsapp/template/list
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| phone_number_id | String | Sim | Número de telefone da conta do WhatsApp ID |

## Exemplos de Requisição

### <Icon icon="circle-right" color="#34D399" size={20} /> Requisição GET

```bash
https://app.wizebot.com.br/api/v1/whatsapp/template/list?apiToken=API-KEY&phone_number_id=PHONE-NUMBER-ID
```

### <Icon icon="circle-right" color="#F87171" size={20} /> Requisição POST

```bash
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/template/list' \
  -d 'apiToken=API-KEY' \
  -d 'phone_number_id=PHONE-NUMBER-ID'
```

## Observações

- Os templates retornados são apenas os aprovados pelo WhatsApp
- O status pode ser: APPROVED, PENDING, REJECTED
- As categorias podem variar conforme as políticas do WhatsApp
- Os componentes podem ser: HEADER, BODY, FOOTER, BUTTONS
- Suporta paginação através dos cursores na resposta
