# API do WhatsApp Wizebot

A API do WhatsApp do Wizebot permite que você integre facilmente as funcionalidades do WhatsApp Business em sua aplicação. Com nossa API, você pode enviar mensagens, gerenciar contatos, criar campanhas de broadcast e muito mais.

## Autenticação

Todas as requisições à API precisam ser autenticadas usando um `apiToken`. Este token deve ser incluído em todas as chamadas à API.

```bash
apiToken: Sua-Chave-API
```

## Formatos de Resposta

Todas as respostas da API são retornadas no formato JSON. Um exemplo de resposta bem-sucedida:

```json
{
  "status": "1",
  "message": "Operação realizada com sucesso"
}
```

Em caso de erro:

```json
{
  "status": "0",
  "message": "Mensagem de erro específica"
}
```

## Limites de Taxa

Para garantir a estabilidade do serviço, implementamos limites de taxa em nossas APIs. Entre em contato com nosso suporte para mais informações sobre limites específicos para sua conta.

## Recursos Principais

Nossa API oferece os seguintes recursos principais:

- **Gerenciamento de Conta** - Conecte e gerencie contas do WhatsApp Business
- **Envio de Mensagens** - Envie mensagens de texto e templates
- **Gerenciamento de Contatos** - Adicione e gerencie seus contatos
- **Templates** - Gerencie e envie mensagens usando templates
- **Catálogo** - Gerencie seu catálogo de produtos
- **Etiquetas** - Organize seus contatos com etiquetas
- **Equipe** - Gerencie atribuições de chat para sua equipe
