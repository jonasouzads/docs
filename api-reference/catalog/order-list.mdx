title: 'Listar Pedidos do Catálogo'
description: 'Este endpoint permite obter a lista de todos os pedidos do catálogo.'
---

## Endpoint

<Icon icon="circle-right" color="#34D399" size={20} /> **GET** 
<Icon icon="circle-right" color="#F87171" size={20} /> **POST**

```bash
https://app.wizebot.com.br/api/v1/whatsapp/catalog/order/list
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| whatsapp_catalog_id | String | Não | Seu ID de catálogo do WhatsApp |

## Exemplos de Requisição

### <Icon icon="circle-right" color="#34D399" size={20} /> Requisição GET

```bash
https://app.wizebot.com.br/api/v1/whatsapp/catalog/order/list?apiToken=API-KEY&whatsapp_catalog_id=WHATSAPP-CATALOG-ID
```

### <Icon icon="circle-right" color="#F87171" size={20} /> Requisição POST

```bash
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/catalog/order/list' \
  -d 'apiToken=API-KEY' \
  -d 'whatsapp_catalog_id=WHATSAPP-CATALOG-ID'
```
