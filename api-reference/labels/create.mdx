---
title: 'Criar Etiqueta'
description: 'Este endpoint permite criar uma nova etiqueta para sua conta do WhatsApp Business.'
---

## Endpoint

<Icon icon="circle-right" color="#34D399" size={20} /> **GET** 
<Icon icon="circle-right" color="#F87171" size={20} /> **POST**

```bash
https://app.wizebot.com.br/api/v1/whatsapp/label/create
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| apiToken | String | Sim | Sua chave de API |
| phone_number_id | String | Sim | Seu número de telefone do WhatsApp |
| label_name | String | Sim | Nome da etiqueta |

## Exemplos de Requisição

### <Icon icon="circle-right" color="#34D399" size={20} /> Requisição GET

```bash
https://app.wizebot.com.br/api/v1/whatsapp/label/create?apiToken=API-KEY&phone_number_id=PHONE-NUMBER-ID&label_name=LABEL-NAME
```

### <Icon icon="circle-right" color="#F87171" size={20} /> Requisição POST

```bash
curl -X POST \
  'https://app.wizebot.com.br/api/v1/whatsapp/label/create' \
  -d 'apiToken=API-KEY' \
  -d 'phone_number_id=PHONE-NUMBER-ID' \
  -d 'label_name=LABEL-NAME'
```

## Exemplo de Resposta

```json
{
  "status": "1",
  "message": "Label has been created successfully."
}
```

## Observações

- O nome da etiqueta deve ser único
- O ID da etiqueta retornado pode ser usado em outras chamadas da API
- Uma vez criada, a etiqueta pode ser atribuída a múltiplos assinantes
